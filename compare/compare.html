COMPARE
Compare Finite Element Solutions on Different Meshes.

COMPARE is a FORTRAN77 program which compares two finite element solutions to a discretized PDE, which were generated on different meshes.

List of Routines:

  • MAIN is the main program for COMPARE.
  • BSP0 computes linear basis functions associated with pressure.
  • DIF prints the difference between two solutions at every node.
  • EXTEND extends a coarse solution to a finer mesh.
  • INTERV brackets a value between sorted data points.
  • L2DIF computes the L2 norm of the differences between the values
  • LL2DIF computes the little L2 norm of the differences
  • LLINFDIF computes the little L infinity norm of the differences
  • PLOT prints out the solution at the nodes.
  • PLVAL evaluates a piecewise linear function at a given point.
  • PPVALU evaluates a piecewise polynomial function.
  • PQVAL evaluates a piecewise quadratic function at a given point.
  • QBF0 evaluates a quadratic basis function in a nonisoparametric element.
  • READS reads scalar data from a file.
  • READV reads vector data from a file.
  • REFBSP0 evaluates linear basis functions in a reference triangle.
  • REFQBF0 evaluates quadratic basis functions in a reference triangle.
  • UVAL0 evaluates the velocities and pressure in a given element.
  • XOFXSI computes the (X,Y) coordinates of a (XSI,ETA) reference.
