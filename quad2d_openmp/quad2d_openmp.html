QUAD2D_OPENMP
Estimate an Integral over a 2D Domain using OpenMP

QUAD2D_OPENMP is a FORTRAN77 program which estimates an integral over a 2D rectangle by using an averaging technique, and uses OpenMP to carry out the calculation in parallel.
