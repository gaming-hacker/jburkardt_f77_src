HYPERBALL_MONTE_CARLO
Monte Carlo Estimate of Hyperball Integrals

HYPERBALL_MONTE_CARLO is a FORTRAN77 library which estimates the integral of F(X) over the interior of the unit hyperball in M dimensions.

Languages:

HYPERBALL_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.
