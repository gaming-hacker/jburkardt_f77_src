MESH_TO_ICE
Transfer ICE dataset from MESH format to NETCDF format

MESH_TO_ICE is a FORTRAN77 program which reads a MESH file containing an ICE dataset and rewrites the information as a NETCDF file.

Usage:

mesh_to_ice prefix
where
  • prefix is the filename prefix,
reads prefix.mesh and creates prefix.nc.

Examples and Tests:

HEXAHEXA_2x2x2 subdivides the unit cube into a 2x2x2 mesh of 8 subcubes or "hexahedra", using 27 vertices, and 24 boundary quadrilaterals.

  • hexahexa_2x2x2.mesh, the MESH file.
  • hexahexa_2x2x2.nc, the NETCDF file created by the program.

PART is a large 3D object including 28,694 vertices, 14,890 triangles, 150,779 tetrahedrons, 50 corners, 1905 edges and 1905 ridges.

  • part.mesh, the MESH file.
  • part.nc, the NETCDF file created by the program.
