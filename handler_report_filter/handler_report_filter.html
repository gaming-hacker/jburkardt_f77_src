HANDLER_REPORT_FILTER
Process HANDLER Output files.

HANDLER_REPORT_FILTER is a FORTRAN77 program which "cleans up" the output from a HANDLER file. The resulting file is more suitable for editing with the MRGray program.

Source Code:

  • handler_report_filter.f, the source code.
  • handler_report_filter.csh, commands to compile the source code.

Examples and Tests:

  • may18c.txt, a data file created by HANDLER.
  • may18c_filtered.txt, the data file, after treatement by HANDLER_REPORT_FILTER, using an RGB translation of (14,25,37).

List of Routines:

  • MAIN is the main program for HANDLER_REPORT_FILTER.
  • CAPCHR capitalizes a single character.
  • CHRCTI2 finds and reads an integer from a string.
  • DIGTEN returns the integer value of a base 10 digit.
  • INTNEXRD finds and reads the next integer in a string.
  • LBEGI compares two strings, ignoring blanks and case.
  • LENCHR returns the length of STRING up to the last nonblank.
  • LEQI is a case insensitive comparison of two strings for equality.
  • SORT_HEAP_EXTERNAL externally sorts a list of items into linear order.
  • TIMESTAMP prints out the current YMDHMS date as a timestamp.
