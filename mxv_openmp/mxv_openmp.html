MXV_OPENMP
Matrix Vector Multiplication Timings with OpenMP

MXV_OPENMP is a FORTRAN77 program which sets up several matrix vector multiplication problems y=A*x, and carries them out using

  • "plain vanilla" FORTRAN;
  • "plain vanilla" FORTRAN plus OpenMP parallelization;

The calculations are carried out for a variety of matrix shapes and sizes, and the elapsed wall clock time is measured and reported. The results are up to the user to analyze and explain!

Usage:

In the BASH shell, the program could be run with 2 threads using the commands: 

        export OMP_NUM_THREADS=2
        ./mxv_openmp

Examples and Tests:

MXV_LOCAL runs the program locally.

  • mxv_local.sh, commands to compile the program and run with 1, 2, 4, and 8 threads.
  • mxv_local_output.txt, the output file.

MXV_ITHACA runs the program through ITHACA's batch system.

  • mxv_ithaca.sh, commands to compile the program and run with 1, 2, 4, and 8 threads.
  • mxv_ithaca_output.txt, the output file.

List of Routines:

  • MAIN is the main program for MXV_OPENMP.
  • TEST01 compares various algorithms for a given matrix size MxN.
  • MATGEN generates a random matrix A and vector X.
  • MXV_PLAIN computes y = A * x, using "plain" code.
  • MXV_PLAIN_OPENMP computes y = A * x, using OpenMP parallel directives.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
