EXAMPLE04
CLAWPACK Example

EXAMPLE04 is a 2D problem, a scalar equation of the form 

        u_t + (u^2)_x + (u^4)_y = 0
Piecewise constant data is used.

The user input includes the following:

  • example04.f, a sample calling program.
  • claw2ez.data, the main input data, read by CLAW2EZ.

Here is a set of commands to run the program, and the main output that results:

  • example04.csh, commands to compile and run the sample program.
  • example04.out, the output from a run of the sample program.

Data files are created for each time step, so that plotting can be done:
