F77_SIMPLE
A Simple Makefile Example

F77_SIMPLE is a simple example of how a makefile can be used to manage a set of FORTRAN77 files.

The underlying task is to approximate the integral of a function F(T) using the midpoint rule. The program to do this is broken up into three FORTRAN77 files:

  • f77_simple.f contains the main program, which tries the midpoint rule three times, each time using more intervals.
  • midpoint.f carries out the midpoint rule to approximate the integral of a function.
  • f.f evaluates the function whose approximate integral is desired.

  • makefile contains the makefile for managing these files.
  • f77_simple.out contains the output from a run of the compiled program.
