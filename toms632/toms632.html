TOMS632
The 0-1 Multiple Knapsack Problem

TOMS632 is a FORTRAN77 library which solves the 0-1 multiple knapsack problem.

Languages:

TOMS632 is available in a FORTRAN77 version.

List of Routines:

  • MKP solves a 0-1 multiple knapsack problem.
  • SIGMA computes an upper bound on the best final solution.
  • PI computes a feasible solution to the problem.
  • PAR carries out parametric computation of the upper bounds.
  • SKP solves the 0-1 single knapsack problem.
  • TIMESTAMP prints out the current YMDHMS date as a timestamp.
