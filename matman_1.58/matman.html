MATMAN
The Matrix Manipulator

MATMAN is a FORTRAN77 program which allows an interactive user to define a matrix and carry out the individual steps of certain algorithms.

MATMAN is an interactive matrix manipulator which assists a student to carry out elementary row operations on a matrix. This program was developed by Charles Cullen and John Burkardt.

MATMAN features include:

  • Interactivity: students prescribe each row operation;
  • Accuracy: rational, real or decimal arithmetic may be used;
  • Affordability: the price is right (it's freeware);
  • Portability: available for IBM (DOS or Windows), Mac (68K or PPC), UNIX.
  • Repeatability: A transcript file records all work;
  • Determines if a matrix is in row echelon form;
  • Extensive help menu;
  • Special options for linear programming problems;
  • Improvability: Developers responsive to user requests.
