TOMS291
Approximate Log(Gamma(X))

TOMS291 is a FORTRAN77 library which approximates the logarithm of the Gamma function.

The text of many ACM TOMS algorithms is available online through ACM: http://www.acm.org/pubs/calgo or NETLIB: http://www.netlib.org/toms/index.html.

Languages:

TOMS291 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:

TOMS708, a FORTRAN77 library which evaluates several special functions, including the logarithm of the Gamma function.
