GRAFIK
Some Primitive Graphics Primitives

GRAFIK is a FORTRAN77 library which contains some simple graphics primitives I used in connection with various scientific plotting needs a long time ago.

Related Data and Programs:

ANYPLT a FORTRAN77 library which served as a generic interface between FORTRAN90 programs and various graphics output devices and files.

TTYPLT a FORTRAN77 library which creates simple typewriter graphics images of mathematical data.

List of Routines:

  • COPLOT makes a contour plot of a scalar quantity.
  • DEFSIZ restores the default plot sizes.
  • QUAD divides a quadrilateral into triangles for contour searches.
  • RELIEF interpolates data on a subdivision of a square.
  • SETSIZ allows the user to change the size of the plot.
  • SMOOTH interpolates missing data.
  • TIMESTAMP prints out the current YMDHMS date as a timestamp.
  • TRIANG searches for contour lines in a triangle.
  • VCPLOT plots a vector quantity.
  • BLOCKDATA initializes common block data.
