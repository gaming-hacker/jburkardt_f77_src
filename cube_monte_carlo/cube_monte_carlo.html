CUBE_MONTE_CARLO
Monte Carlo Estimate of Integrals Over Unit Cube in 3D

CUBE_MONTE_CARLO is a FORTRAN77 library which estimates the integral of a function over the interior of the unit cube in 3D.

Languages:

CUBE_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.
